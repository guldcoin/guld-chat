Registering a Username on guld - Invitation Only

Once you have applied for your individual or group username with gai in the telegram chat You will need to sign a 1 guld registration transaction.

Follow these steps to send your first signed transaction to gai.

  1. Send a message to gai in the guld Telegram chat or support group, or in a direct message finding gai in the guld user list in any of the groups he is in.

  2. To register type command /register individual username, where username is your desired name.

  3. Gai will create a transaction like the following for you to sign.

2018/03/06 * register individual
    ; timestamp: 1520397978
    username:Assets   -1 GULD
    username:Expenses:guld:register   1 GULD
    guld:Liabilities   1 GULD
    guld:Income:register:individual:username   -1 GULD
